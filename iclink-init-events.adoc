== iclink.init.error
*Severity*::
ALERT
*Description*::
This message occurs when the HA interconnect fails to initialize.
*Corrective Action*::
Reboot the system. If the problem persists, run diagnostics on the memory in the node.
*Syslog Message*::
(None).
*Parameters*::
*errorCode* (INT): The internal error code.
