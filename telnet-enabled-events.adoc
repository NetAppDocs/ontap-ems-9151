== telnet.enabled
*Severity*::
NOTICE
*Description*::
This message occurs when telnet is enabled in the cluster using the (privilege: advanced) "security protocol" command.
*Corrective Action*::
(None).
*Syslog Message*::
Insecure protocol service telnet has been enabled in the cluster.
*Parameters*::
(None).
