== aggr.full
*Severity*::
ALERT
*Description*::
This message occurs when an aggregate is full.
*Corrective Action*::
Ensure that all containers in the pod are online and healthy. Check if the REST server is up and responding.
*Syslog Message*::
Aggregate "%s" is full.
*Parameters*::
*aggr_name* (STRING): Name of the aggregate.
