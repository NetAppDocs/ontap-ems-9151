== geom.ontap.orphan.removing
*Severity*::
NOTICE
*Description*::
This message occurs when the geometry layer (geom) detects a device to be detached from the system.
*Corrective Action*::
(None).
*Syslog Message*::
Removing unit %d type %d.
*Parameters*::
*unit* (INT): Unit number.
*disk_type* (INT): Disk type.
