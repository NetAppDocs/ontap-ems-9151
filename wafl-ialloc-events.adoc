== wafl.ialloc.is.disabled
*Severity*::
NOTICE
*Description*::
This message occurs when an inode allocation in a stripe is disabled.
*Corrective Action*::
(None).
*Syslog Message*::
Inode allocation in stripe is disabled for %s inode space.
*Parameters*::
*ino_type* (STRING): Inode type.
