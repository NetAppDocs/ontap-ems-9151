== sfu.suspendDiskIO
*Severity*::
INFORMATIONAL
*Description*::
This message occurs when a disk shelf firmware update is started and disk I/O is suspended.
*Corrective Action*::
(None).
*Syslog Message*::
Suspending disk I/O to reboot shelf modules for shelf firmware update.
*Parameters*::
(None).
