== snapmirror.sockStrWriteErr
*Severity*::
ERROR
*Description*::
This event is issued when SnapMirror tries to write a large string to the socket.
*Corrective Action*::
(None).
*Syslog Message*::
Write string too large %d.
*Parameters*::
*len* (INT): String length SnapMirror tried to write to socket.
