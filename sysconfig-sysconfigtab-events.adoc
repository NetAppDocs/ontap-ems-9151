== sysconfig.sysconfigtab.openFailed
*Severity*::
NOTICE
*Description*::
This event is issued when the sysconfigtab file cannot be opened.
*Corrective Action*::
(None).
*Syslog Message*::
sysconfig: %s
*Parameters*::
*reason* (STRING): Reason why it can't be opened.
