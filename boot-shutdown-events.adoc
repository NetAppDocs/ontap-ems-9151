== boot.shutdown.issue
*Severity*::
ALERT
*Description*::
This message occurs when an error is detected while shutting down the system.
*Corrective Action*::
Contact NetApp technical support.
*Syslog Message*::
Failed to shutdown the system. Error: %s
*Parameters*::
*err_str* (STRING): Error string.
