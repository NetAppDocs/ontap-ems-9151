== configbr.backupCompleted
*Severity*::
NOTICE
*Description*::
This message occurs when a scheduled configuration backup is successfully created.
*Corrective Action*::
(None).
*Syslog Message*::
Scheduled configuration backup %s was created successfully.
*Parameters*::
*backup* (STRING): Backup file.
