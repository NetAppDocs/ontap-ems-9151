== lun.serial.timeWentBack
*Severity*::
INFORMATIONAL
*Description*::
This event occurs when the system clock has been set back in time. It prints the current LUN serial number counter.
*Corrective Action*::
(None).
*Syslog Message*::
(None).
*Parameters*::
*counter* (LONGINT): Current value of LUN serial number counter.
