== nvram.parity.threshold
*Severity*::
ERROR
*Description*::
This message occurs when excessive parity errors are reported by the NVRAM complex. Your system is at risk of data loss in the event of a power failure.
*Corrective Action*::
Contact NetApp technical support.
*Syslog Message*::
Excessive parity errors detected in the NVRAM module.
*Parameters*::
(None).
