== coredump.compress.failed
*Severity*::
ERROR
*Description*::
An error was encountered while compressing data for the coredump. This core will be deleted.
*Corrective Action*::
(None).
*Syslog Message*::
Error compressing core data: %s
*Parameters*::
*msg* (STRING): The error message
