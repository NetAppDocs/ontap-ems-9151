== fabric.sdm.mem.alloc
*Severity*::
NOTICE
*Description*::
This message occurs when the system fails to allocate a message for storage device monitor (SDM) notification.
*Corrective Action*::
(None).
*Syslog Message*::
Failed to allocate a message for SDM notification to fabric_man.
*Parameters*::
(None).
