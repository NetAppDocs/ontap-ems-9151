== zapi.killed
*Severity*::
NOTICE
*Description*::
This message occurs when ZAPI Socket Manager (ZSM) terminates and no longer offers a connection on which a client can deliver Zephyr API (ZAPI) requests.
*Corrective Action*::
(None).
*Syslog Message*::
ZAPI Socket Manager (ZSM) terminated.
*Parameters*::
(None).
