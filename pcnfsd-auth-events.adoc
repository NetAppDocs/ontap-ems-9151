== pcnfsd.auth.failure
*Severity*::
ERROR
*Description*::
This message occurs when the authorization for a user fails due to a corrupted password entry in /etc/passwd.
*Corrective Action*::
Restore the password entry for the user.
*Syslog Message*::
Corrupted password in /etc/passwd file.
*Parameters*::
(None).
