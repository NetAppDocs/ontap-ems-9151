== mcc.encryption.disabled
*Severity*::
NOTICE
*Description*::
This message occurs when the encryption for NVLog and storage data in a MetroCluster (tm) configuration is disabled.
*Corrective Action*::
(None).
*Syslog Message*::
The MetroCluster encryption feature has been disabled.
*Parameters*::
(None).
