== iscsi.notice
*Severity*::
NOTICE
*Description*::
This event is issued when a loggable, non-error event occurs within the iSCSI subsystem.
*Corrective Action*::
(None).
*Syslog Message*::
ISCSI: %s
*Parameters*::
*m* (STRING): A more specific message describing the event.
