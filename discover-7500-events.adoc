== discover.7500.supported
*Severity*::
NOTICE
*Description*::
This message occurs when discovery of a 7500N FC-to-SAS bridge is enabled.
*Corrective Action*::
(None).
*Syslog Message*::
FC-to-SAS bridge ATTO 7500N discovery is enabled.
*Parameters*::
(None).

*Severity*::

*Description*::

*Corrective Action*::
(None).
*Syslog Message*::
FC-to-SAS bridge ATTO 7500N discovery is disabled.
*Parameters*::
(None).
