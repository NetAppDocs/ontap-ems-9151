== air.qtm.rebuild.scan.state
*Severity*::
NOTICE
*Description*::
This message occurs when the state of a scanner used to rebuild the qtree metafile changes.
*Corrective Action*::
(None).
*Syslog Message*::
Scanner to rebuild the qtree metafile is %s.
*Parameters*::
*state* (STRING): State of the scanner.
