== mct.control.error
*Severity*::
NOTICE
*Description*::
This message occurs when the Mirror Cache Transport layer encounters an error in its control group.
*Corrective Action*::
(None).
*Syslog Message*::
Mirror Cache transport control error: '%s'.
*Parameters*::
*error_message* (STRING): Details for the encountered error.
