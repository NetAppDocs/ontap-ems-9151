== sfu.resumeDiskIO
*Severity*::
INFORMATIONAL
*Description*::
This message occurs when a disk shelf firmware update is completed and disk I/O is resumed.
*Corrective Action*::
(None).
*Syslog Message*::
Resuming disk I/O after shelf firmware update.
*Parameters*::
(None).
