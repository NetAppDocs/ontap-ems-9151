== config.NoSwitchCount
*Severity*::
NOTICE
*Description*::
This message occurs when an attempt to obtain the switch port count fails because it does not exist.
*Corrective Action*::
(None).
*Syslog Message*::
Unable to obtain switch port count information for attached switches.
*Parameters*::
(None).
