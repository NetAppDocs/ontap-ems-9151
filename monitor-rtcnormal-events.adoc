== monitor.rtcNormal
*Severity*::
NOTICE
*Description*::
This message occurs when the real-time clock (RTC) battery voltage is normal. Previously occurred fault is now cleared.
*Corrective Action*::
(None).
*Syslog Message*::
Real-time clock battery voltage is normal.
*Parameters*::
(None).
