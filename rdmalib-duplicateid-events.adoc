== rdmalib.duplicateId
*Severity*::
ALERT
*Description*::
This message occurs when the local node system identifier matches the partner node's system identifier.
*Corrective Action*::
Contact NetApp technical support.
*Syslog Message*::
Interconnect : Partner ID %u is the same as our id.
*Parameters*::
*id* (INT): System Id
