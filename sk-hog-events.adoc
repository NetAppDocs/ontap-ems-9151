== sk.hog.runtime
*Severity*::
NOTICE
*Description*::
This event is issued when a process runs too long.
*Corrective Action*::
(None).
*Syslog Message*::
Process %s ran for %d milliseconds
*Parameters*::
*procname* (STRING): The name of the process which ran too long
*runtime* (INT): Number of milliseconds process ran
