== oauth2.status.disabled
*Severity*::
NOTICE
*Description*::
This message occurs when the OAuth 2.0 status of the cluster is changed to DISABLED.
*Corrective Action*::
(None).
*Syslog Message*::
Status of OAuth 2.0 has been changed to DISABLED.
*Parameters*::
(None).
