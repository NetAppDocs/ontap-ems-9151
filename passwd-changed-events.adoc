== passwd.changed
*Severity*::
NOTICE
*Description*::
The password for a user has been changed.
*Corrective Action*::
(None).
*Syslog Message*::
passwd for user '%s' changed.
*Parameters*::
*username* (STRING): The name of the user whose password has just been changed.
