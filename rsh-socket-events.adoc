== rsh.socket.failure
*Severity*::
ERROR
*Description*::
The rshd daemon could not create a socket.
*Corrective Action*::
Disable and enable rshd daemon using options rsh.enable.
*Syslog Message*::
RSH socket error: %s.
*Parameters*::
*reason* (STRING): The reason why the rshd daemon could not create a socket.
