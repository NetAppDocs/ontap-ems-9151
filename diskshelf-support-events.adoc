== diskShelf.support.file
*Severity*::
NOTICE
*Description*::
This message occurs when a new disk shelf support file is detected. Current support level of existing shelves will be checked.
*Corrective Action*::
(None).
*Syslog Message*::
New disk shelf support file (DQP) detected.
*Parameters*::
(None).
