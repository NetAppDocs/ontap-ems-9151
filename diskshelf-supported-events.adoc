== diskShelf.supported
*Severity*::
NOTICE
*Description*::
This message occurs when a newly supported disk shelf is detected.
*Corrective Action*::
(None).
*Syslog Message*::
Disk shelf %s (%s) is now supported.
*Parameters*::
*device* (STRING): Device address of the disk shelf.
*product_id* (STRING): Product ID of the disk shelf.
