== nv.none
*Severity*::
EMERGENCY
*Description*::
This message occurs when the system cannot detect a nonvolatile (NV) device.
*Corrective Action*::
Contact NetApp technical support.
*Syslog Message*::
No NV device has been detected.
*Parameters*::
(None).
