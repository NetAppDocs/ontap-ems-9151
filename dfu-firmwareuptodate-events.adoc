== dfu.firmwareUpToDate
*Severity*::
NOTICE
*Description*::
This message occurs when all eligible disks are determined to be updated to the latest version of firmware available on the system.
*Corrective Action*::
(None).
*Syslog Message*::
Firmware is up-to-date on all eligible disks.
*Parameters*::
(None).
