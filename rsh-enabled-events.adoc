== rsh.enabled
*Severity*::
NOTICE
*Description*::
This message occurs when rsh is enabled in the cluster using the (privilege: advanced) "security protocol" command.
*Corrective Action*::
(None).
*Syslog Message*::
Insecure protocol service rsh has been enabled in the cluster.
*Parameters*::
(None).
