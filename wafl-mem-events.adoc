== wafl.mem.stomper
*Severity*::
NOTICE
*Description*::
This event indicates whether memory stomper code is running or not.
*Corrective Action*::
(None).
*Syslog Message*::
You are %s running in corruption detection diagnostic mode with degraded performance.
*Parameters*::
*onoff* (STRING): The stomper code is on/off
