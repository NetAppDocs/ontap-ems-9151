== iscsi.warning
*Severity*::
NOTICE
*Description*::
This event is issued if the iSCSI subsystem detects an abnormal state.
*Corrective Action*::
(None).
*Syslog Message*::
ISCSI: %s
*Parameters*::
*m* (STRING): A more specific message describing the state.
