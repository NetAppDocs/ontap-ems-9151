== savecore.readonly
*Severity*::
NOTICE
*Description*::
An attempt to run the savecore command was made during a read-only boot. This is not permitted.
*Corrective Action*::
(None).
*Syslog Message*::
Core dumps not saved in last resort read-only mode
*Parameters*::
(None).
