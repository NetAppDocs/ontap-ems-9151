== malloc.failure
*Severity*::
ERROR
*Description*::
This event is generated if malloc is called unsafely.
*Corrective Action*::
(None).
*Syslog Message*::
%s %s %s %s %s %s %s %s %s %s %s

*f5* (STRING): Sixth frame of backtrace.
*f6* (STRING): Seventh frame of backtrace.
