== raid.switchoverNoDisks
*Severity*::
INFORMATIONAL
*Description*::
This message occurs when a MetroCluster(tm) switchover finds no disks during RAID assimilation.

(None).
*Syslog Message*::
MetroCluster switchover assimilation found no disks.

(None).
