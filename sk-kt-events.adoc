== sk.kt.rtc
*Severity*::
INFORMATIONAL
*Description*::
This is the output of debugging command that prints the time of day held in the real-time clock chip.
*Corrective Action*::
(None).
*Syslog Message*::
RTC time: %s
*Parameters*::
*rtc_time* (STRING): The time held in the real-time clock chip.
